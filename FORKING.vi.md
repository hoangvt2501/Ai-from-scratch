# Hướng dẫn phân nhánh

Khóa học này được MIT cấp phép. Bạn có thể tự do phân nhánh và điều chỉnh nó cho phù hợp với nhu cầu của mình. Đây là cách để làm điều đó tốt.

## Dành cho nhóm

Bạn muốn sử dụng điều này làm training nội bộ? Fork và tùy chỉnh:

1. Ngã ba repository
2. Loại bỏ các giai đoạn mà nhóm của bạn không cần
3. Thêm các ví dụ và dữ liệu cụ thể của công ty
4. Thêm tích hợp công cụ nội bộ vào đầu ra
5. Giữ phân bổ - nó giúp cộng đồng phát triển

## Dành cho trường học và đại học

Bạn muốn sử dụng điều này làm tài liệu khóa học?

1. Ngã ba repository
2. Lập bản đồ các giai đoạn với lịch trình học kỳ của bạn
3. Thêm bảng đánh giá chấm điểm vào bài tập
4. Thêm bài tập và bài kiểm tra của riêng bạn
5. Cân nhắc đóng góp cải tiến ngược dòng

## Đối với Bootcamp

Điều hành một bootcamp trả phí? Điều đó tốt theo MIT.

1. Phân nhánh và cấu trúc cho dòng thời gian nhóm thuần tập của bạn
2. Thêm nội dung video, sessions trực tiếp, cố vấn
3. Mã và tài liệu là của bạn để xây dựng
4. Cân nhắc tài trợ cho dự án hoặc đóng góp lại

## Đối với các ngôn ngữ khác

Bạn muốn dạy chương trình giảng dạy này bằng một ngôn ngữ lập trình khác?

1. Ngã ba repository
2. Triển khai lại các ví dụ về mã bằng ngôn ngữ của bạn
3. Giữ nguyên cấu trúc bài học và tài liệu
4. Gửi PR để liên kết fork của bạn từ README chính

## Luôn cập nhật fork của bạn

```bash
git remote add upstream https://github.com/rohitg00/ai-engineering-from-scratch.git

git fetch upstream
git merge upstream/main
```
